php基础语法学习Ⅰ

1.php脚本以结束。且php文件中通常包含HTML标签。
2.php中有两种在浏览器中输出文本的基础指令：echo和print。
3.php中注释与C语言中相同。
4.PHP 变量规则：
变量以 $ 符号开始，后面跟着变量的名称。
变量名必须以字母或者下划线字符开始（与C语言一致）。
变量名只能包含字母、数字以及下划线，不能包含空格（与C语言一致）。
变量名是区分大小写的（$y 和 $Y 是两个不同的变量）。
5.php中没有声明变量的命令，变量将在你第一次给他赋值时被创建。
6.php中可以赋一个文本值（类似C语言中字符串）给变量，需要在文本
值两侧加上引号。
7.PHP共有四种不同的变量作用域：local，global，，static，parameter。
8.在所有函数外部定义的变量，拥有全局作用域。除了函数外，全局变量可
以被脚本中的任何部分访问，要在一个函数中访问一个全局变量，需要使
用 global 关键字。在 PHP 函数内部声明的变量是局部变量，仅能在函数
内部访问。在不同函数中可以使用同样的变量名，因为这些变量名只是局
部变量。
9.php将所有全局变量储存在一个名为$GLOBALS[]的数组中，为保存变量
的名称。这个数组可以在函数内部被访问。
10.php中当一个函数完成时，他的所以变量通常会被删除，如果希望某个局
部变量不被删除，可以在第一次声明这个变量时在前面加上static。此后每次
调用该函数时，改变量将会保存函数前一次被调用时的值，如：

但是要注意改函数仍是局部变量。
11.参数是通过调用代码将值传递给函数的局部变量。参数是在参数列表中声明
的，作为函数声明的一部分。

burp的基本模块的初步认识

proxy
此模块是burp的核心，也是使用最多的。其核心功能为拦截并修改http数据包。
在burp中数据是互通的，所以在proy中单击右键把数据包传到其他模块中。
repeater
当遇到需要探索HTTP Haeder中的注入点和POST数据包中的SSRF漏洞时需要使
用到repeater重放模块。本模块由左右两个子模块组成，左边是初始的HTTP请求
，右侧为服务器返回的数据。在左侧我们可以对数据进行修改，然后点击go就可
以在右侧得到服务器反馈的数据。
interuder
Intruder模块可以提供简便的暴力破解功能，可以对网站账号密码等进行爆破测
试。此模块包括target，position，payload，options四个标签页：
1.target
可以设置攻击地址和目标端口，并选择是否使用https协议。
2.position
在position标签页可以选择本次攻击的位置和攻击方法。攻击位置可以自动选择，
也可以手动选择。自动选择模式下，Burp会选择很多变量，比较费时，一般都采
用手动模式。手动模式选择时，先点击 Clear$/$清除 清除变量，然后鼠标选择需
要爆破的变量，并点击Add$/添加$按钮即可。在position共有四种攻击方式可选
择，分别为狙击手sniper,破城槌battering ram,音叉pitchpork，集束炸弹cluster
bomb。
（1）狙击手Sniper型只需要设置一个 payload set，在两个变量的位置逐一替换
payload，每次替换一个位置，先替换前面再替换后面。
（2）破城槌Battering ram型 只需要设置一个payload set 在两个变量位置同时
替换相同的payload。
（3)音叉Pitchfork型需要设置两个 payload set 两个变量的位置和两个 payload
set 一一对应。
（4）集束炸弹Cluster bomb 型 需要设置两个 payload set ，这时候每个位置的
payload将在 payload set 中进行排列组合。
3.payload
在payload 标签页，可以设置每一个位置的payload set。 payload type 可以选
择payload 类型，常见的有Runtime file（从文件中选择加载 payloda）、Num
-bers（设置数字的开始和结束）、Dates（设置日期和格式）以及 Character bl
-ocks（设置长度爆破）等。
4.options
在Options 标签页中可以对请求引擎/Request Engine中的参数进行选择，包括线
程数量、网络连接失败时的重传次数、重传前暂停时间、数据包发送速度等内容。
decoder
此模块提供了丰富的编码和解码工具，可以对于数据包进行编码和解码，支持文本
格式和十六进制格式查看。使用时，先将数据放到上面文本框中，点击解码/Deco
-de as 和 编码/Encode as下来菜单选择需要的编码即可。
compare
此模块可以对于两个数据包进行准确识别，并用不同颜色标记出差距。可以从剪切
板中粘贴，也可以从文件导入。

关于指针浅学的一些事项

在使用简接寻址运算符时，若指针变量p没有初始化，不要使用p的值。会使程序报错。
当然更不要试图对p进行赋值，严重的可能导致系统崩溃。
当指针遇上const时有两种情况：
若const在前，如const intp=&i；指的是无法通过p对i进行赋值，而可以对p进行运算
，对i进行赋值。
若·const在后，如intconst p=&i；指的是无法对p进行运算，但可以通过p对i进行赋值。
优先级要高于+但低于++，所以（p+1）需要括号而p++不需要。
指针不能进行乘除运算。
0地址通常表示特殊的地址，它通常是个不能随便碰的地址，你的指针中不应该具有0值。
可以用NULL表示0地址。
指向不同类型的指针不能直接相互赋值。
void*表示不知道指向什么类型的指针。

数组与指针

数组变量其实是特殊的指针，数组变量本身表达地址，如 int a[10];int*p=a;即无需使用
&取地址。但是数组的单元表达的是变量，取地址时仍需要用&。
所以[]既可以对数组使用也可以对指针使用，即p[0]==p,同理，也既可以对指针使用也
可以对数组使用。
实际上数组变量是const的指针，所以数组无法被赋值，就解释了之前学习是所说的不能
将一个数组中每个元素的值赋予另一个数组的各个元素。
因为数组变量已经是const的指针了，所以当出现const int a[]={1，2，3，4，5}；时，
表示的是数组中的每个单元都是const int，所以必须通过初始化进行赋值。因为把数组传
进函数时传递的是地址，所以可以通过将参数设为const来保护数组内的单元的值不会被函
数破环，例 int sum（const int a[],int length);。

关于栈的基本认知

1.栈可以储存具有相同数据类型的多个数据项。
2.栈的操作是受限的，只能往栈中压入数据项（把数据加到栈顶）和从栈中弹出数据项（从同一端移走数据项）。禁止测试或修改不在栈顶的数据项。
即使用栈存储数据元素，对数据元素的“存”和“取”有严格的规定：数据按一定的顺序存储到栈中，当需要调取栈中某数据元素时，需要将在该数
据元素之后进栈的先出栈，该数据元素才能从栈中提取出来。例如栈中存放了 4 个数据元素，进栈的顺序是 A 先进栈，然后 B 进，然后 C 进，最后
D进栈；当需要调取 A 时，首先 D 出栈，然后 C 出栈，然后 B 出栈，最后 A 才能出栈被调用。
3.栈有两种类型顺序栈和链栈。顺序栈通过数组实现，链栈用线性表的链式存储结构实现。
4.栈存储结构调取栈中数据元素时，要避免出现“上溢”和“下溢”的情况：“上溢”：在栈已经存满数据元素的情况下，如果继续向栈内存入数据，
栈存储就会出错。“下溢”：在栈内为空的状态下，如果对栈继续进行取数据的操作，就会出错。但是链栈不会出现”上溢“。

利用递归实现快速排序

近日在课上学习了冒泡排序法，联想到最近学习的函数的递归，
于是利用递归写了一个排序程序，代码如下：
#include<stdio.h>
#define N 10
void sort (int a[], int low, int high);
int split (int a[],int low,int high);

int main()
{
int a[N],i;
for(i=0;i<N;i++)
scanf(“%d”,&a[i]);

sort(a,0,N-1);

for(i=0;i<N;i++)
    printf(" %d",a[i]);
printf("\n");

return 0;

}

void sort (int a[],int low,int high)
{
int middle;

if (low>high)
return;

middle=split(a,low,high);


sort(a,low,middle-1);
sort(a,middle+1,high);

}

int split(int a[],int low,int high)
{
int part_element=a[low];

for(;;){
    
    while(low<high&&part_element<=a[high])
        high--;
    
    if(low>=high)
    break;
    
    a[low]=a[high];
    low++;
    while(low<high&&a[low]<=part_element)
        low++;
        
    if(low>=high)
    break;
    
    a[high]=a[low];
    high--;
}

a[high]=part_element;
return high;

}
代码定义了排序函数sort和分割数列函数split，先通过split得出
一个元素，利用这个元素将函数分割成两段，然后利用递归在两
段进行排序。

C语言函数学习中的一些注意事项

1.函数形式参数为一维数组时可以用第二个参数来提供长度，
如int sum(int a[],int b)但为多维数组时就必须指定除第一维
外其他维度的数量。同时若在多维参数声明中使用static，st
-atic仅可用于第一维。
2.复合字面量：复合字面量是通过指定其包含的元素而创建的
没有名字的数组，格式为先在一对括号内给定类型名，随后在
一对花括号内设定所包括的元素，如 total = sum ((int []) {5,
3,6,9,11},5);
3.若没有给出表达式return可以在返回类型为void的函数中出
现。

title: C语言提高效率的一些方法
date: 2021-10-27 13:22:01
tags:—
1.定义时节约内存，如int会占用4字节的内存而short只会占用2字节的内存，若无必要
尽量节约内存，这样可以提高运行效率。
2.运算时尽量减少运算步骤，根据实际情况减少运算步骤来提高效率。
3.当switch语句中的case标号很多时，为了减少比较的次数，可以使用嵌套switch语句。
把发生频率高的case标号放在一个switch语句中，并且是嵌套switch语句的最外层，发
生相对频率相对低的case标号放在另一个switch语句中。
4.当返回的函数值不需要被用到，可以使用void使函数值不被返回，可以提高程序效率。
5.使用增量和减量操作符。在使用到加一和减一操作时尽量使用增量和减量操作符，效率
要高于赋值语句。
