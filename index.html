php基础语法学习Ⅳ

1.php中的foreach循环
foreach循环用于遍历数组，其格式为：
foreach（$array as $value)
{
；
}
其中array为数组名称，为每次循环要执行的代码。此时每进行一次
循环，当前元素的值就会被赋值给value，然后下一次循环时数组就会
往后一位。
还有：
foreach（$array as $key =>$value)
{
***;
}
此时每进行一次循环，当前元素的值就会被赋值给value而键会被赋给
key，然后下一次循环时数组就会往后一位。适用于关联数组。
2.php中函数
创建函数的格式如下：
function functionname（）
{
；
}
其中functionname为函数名，为要执行的代码。
注意：函数名称不能以数字开头，应该以字母或下划线开头，而且为了
便于审计代码函数名应提示出它的功能。
若需要函数返回一个值，可以使用return语句。

3.php中的魔术常量
php中有8个魔术常量，他们的值随他们在代码中的位置改变而改变。
这些特殊的常量不区分大小写。
_line_文件中当前的行数。
_file_文件的完整路径和文件名。
dir_文件所在的目录，它等价于dirname（_file)。
_function_函数名称。
_class_类的名称。
_trait_trait的名称。
_method_类的方法名。
_namespace_当前命名空间的名称。
4.php中的命名空间
默认情况下，所有常量、类和函数名都放在全局空间下，我们可以使
用namespace声明命名空间，其语法如下：
namespace MyProject {
const CONNECT_OK = 1;
class Connection { /* … / }
function connect() { / … / }
}
如果一个文件中包含命名空间，它必须在所有其他代码前声明命名空
间。在声明命名空间之前唯一合法的代码是用于定义源文件编码方式
的 declare 语句。所有非 PHP 代码包括空白符都不能出现在命名空
间的声明之前。当然，我们也可以在同一个文件中定义不同的命名空
间代码。将全局的非命名空间中的代码与命名空间中的代码组合在一
起，只能使用大括号形式的语法。全局代码必须用一个不带名称的na
-mespace 语句加上大括号括起来。
命名空间的名字可以使用分层次的方式定义,语法如下：
namespace MyProject\Sub\Level;
const CONNECT_OK = 1;
class Connection { / … / }
function Connect() { / … */ }
PHP中命名空间可以解决以下两类问题：
（1）用户编写的代码与PHP内部的类/函数/常量或第三方类/函数/常
量之间的名字冲突。
（2）为很长的标识符名称(通常是为了缓解第一类问题而定义的)创建
一个别名（或简短）的名称，提高源代码的可读性。

php基础语法学习Ⅲ

1.php中的松散比较和严格比较
松散比较为==，它只比较值不比较类型。严格比较为===，除了比较值，它也比较
类型。
2.php中常量
在php中设置常量一般使用define（）函数。格式为define（$name,$value[,$case
_insensitive=false])其中name为常量名称，value为常量的值，case_insensitive为
可选参数，若设为true，则该函数大小写不敏感，默认为大小写敏感。
3.PHP 中，只有一个字符串运算符：并置运算符 . 用于把两个字符串值连接起来。
4.php中若要知道字符串的长度，可使用strlen（）函数。
5.php中我们可以使用strpos（）函数在字符串中查找一个字符或一段文本。若在字
符串中找到匹配，函数会返回第一个匹配的字符位置。若未找到会返回FALSE。
注意：字符串第一个字符的位置时0而不是1。
6.php中的算数运算符、赋值运算符、递增递减运算符与条件运算符都与C语言相同。
7.比较运算符中x<>y为不等于相当于x!=y，x!==y为绝对不等于，表示除了比较值还
需比较类型。
8.逻辑运算符中x xor y为异或，表示如果x和y中有且仅有一个为true，则返回true。
9.php7+中支持组合比较符<=>，它可以轻松实现两个变量的比较，且不局限与数值
类数据的比较。例子如下：
$c=$<a=>$b;
表示若$a>$b，则$c值为1。若$a==$b,则$c的值为0。若$a<$b,则$c的值为-1。
10.$c=&$a;表示c为a的引用，即c和a为同一个变量了。其中&时引用的意思，php中
没有指针的概念。
11.php中数组
php中使用array（）函数创建数组。
php中可以使用count（）函数返回数组的元素数量。
php中有三种类型的数组：数值数组、关联数组、多维数组。其中数值数组与C语言中
的数组类似。
关联数组是使用你分配个数组的指定的键的数组，创建关联数组的方法如下：
$num=array(“a”=>”1”,”b”=>”2”,”c”=>”3”);
此时echo $num[‘a’];会输出1。
12.php中数组排序函数
sort（），将数组中元素按字母升序排列。
rsort（），将数组中元素按字母降序排列。
asort（），将关联数组中元素按数组的值升序排列。
arsort（），将关联数组中元素按数组的值降序排列。
ksort（），将关联数组中的元素按数组的键的字母升序排列。
krsort（），将关联数组中的元素按数组的键的字母降序排列。

php基础语法学习Ⅱ

1.php中有两种基本的输出方式：echo和print。echo可以输出一个或多个字符串，
而print只允许输出一个字符串，返回值总为1。 echo输出的速度比print快，但是
他没有返回值。
2.php EOF
php EOF为一种定义字符串的方法。在编程中有时需要输出大段的html或javascript
脚本，这时如果用传统的字符串输出的话就需要使用大量的转义字符对特殊符号进
行转义。这时就可以使用php定界符eof，它的作用就是按照原样（包括换行格式什
么的）输出其内部的内容。在php定界符eof中任何特殊字符都不需要转义。
*php EOF以<<<EOF开始标记开始，以EOF结束标记结束。结束标记必须顶格写，不
能有缩进和空格，且在结束标记末尾要有分号。
*EOF可以用任意其他字符代替，但是需要结束标记与开始相同且不在正文中出现。
*开始标识可以不带引号或带单双引号，不带引号与带双引号都解释内嵌的变量和转
义符号即此时eof中的变量会用其值代替，带单引号则不解释内嵌的变量和转义符号。
*位于开始标记和结束标记之间的变量可以被正常解析，但是函数则不可以。在 here
-doc 中变量不需要用连接符 . 或 , 来拼接。
3.php中整型可以用三种格式来指定：十进制，十六进制（以 0x 为前缀）或八进制
（前缀为0）。
4.php中NULL 值表示变量没有值。NULL 是数据类型为 NULL 的值。NULL 值指明
一个变量是否为空值。数据空值和NULL值的值相等，但是类型不同，空值为string
NULL值为NULL。我们可以通过设置变量值为 NULL 来清空变量数据。

变量以 $ 符号开始，后面跟着变量的名称。

内部访问。在不同函数中可以使用同样的变量名，因为这些变量名只是局
部变量。

的名称。这个数组可以在函数内部被访问。

部变量不被删除，可以在第一次声明这个变量时在前面加上static。此后每次
调用该函数时，改变量将会保存函数前一次被调用时的值，如：

但是要注意改函数仍是局部变量。
11.参数是通过调用代码将值传递给函数的局部变量。参数是在参数列表中声明
的，作为函数声明的一部分。

burp的基本模块的初步认识

proxy
此模块是burp的核心，也是使用最多的。其核心功能为拦截并修改http数据包。
在burp中数据是互通的，所以在proy中单击右键把数据包传到其他模块中。
repeater
当遇到需要探索HTTP Haeder中的注入点和POST数据包中的SSRF漏洞时需要使
用到repeater重放模块。本模块由左右两个子模块组成，左边是初始的HTTP请求
，右侧为服务器返回的数据。在左侧我们可以对数据进行修改，然后点击go就可
以在右侧得到服务器反馈的数据。
interuder
Intruder模块可以提供简便的暴力破解功能，可以对网站账号密码等进行爆破测
试。此模块包括target，position，payload，options四个标签页：
1.target
可以设置攻击地址和目标端口，并选择是否使用https协议。
2.position
在position标签页可以选择本次攻击的位置和攻击方法。攻击位置可以自动选择，
也可以手动选择。自动选择模式下，Burp会选择很多变量，比较费时，一般都采
用手动模式。手动模式选择时，先点击 Clear$/$清除 清除变量，然后鼠标选择需
要爆破的变量，并点击Add$/添加$按钮即可。在position共有四种攻击方式可选
择，分别为狙击手sniper,破城槌battering ram,音叉pitchpork，集束炸弹cluster
bomb。
（1）狙击手Sniper型只需要设置一个 payload set，在两个变量的位置逐一替换
payload，每次替换一个位置，先替换前面再替换后面。
（2）破城槌Battering ram型 只需要设置一个payload set 在两个变量位置同时
替换相同的payload。
（3)音叉Pitchfork型需要设置两个 payload set 两个变量的位置和两个 payload
set 一一对应。
（4）集束炸弹Cluster bomb 型 需要设置两个 payload set ，这时候每个位置的
payload将在 payload set 中进行排列组合。
3.payload
在payload 标签页，可以设置每一个位置的payload set。 payload type 可以选
择payload 类型，常见的有Runtime file（从文件中选择加载 payloda）、Num
-bers（设置数字的开始和结束）、Dates（设置日期和格式）以及 Character bl
-ocks（设置长度爆破）等。
4.options
在Options 标签页中可以对请求引擎/Request Engine中的参数进行选择，包括线
程数量、网络连接失败时的重传次数、重传前暂停时间、数据包发送速度等内容。
decoder
此模块提供了丰富的编码和解码工具，可以对于数据包进行编码和解码，支持文本
格式和十六进制格式查看。使用时，先将数据放到上面文本框中，点击解码/Deco
-de as 和 编码/Encode as下来菜单选择需要的编码即可。
compare
此模块可以对于两个数据包进行准确识别，并用不同颜色标记出差距。可以从剪切
板中粘贴，也可以从文件导入。

关于指针浅学的一些事项

在使用简接寻址运算符时，若指针变量p没有初始化，不要使用p的值。会使程序报错。
当然更不要试图对p进行赋值，严重的可能导致系统崩溃。
当指针遇上const时有两种情况：
若const在前，如const intp=&i；指的是无法通过p对i进行赋值，而可以对p进行运算
，对i进行赋值。
若·const在后，如intconst p=&i；指的是无法对p进行运算，但可以通过p对i进行赋值。
优先级要高于+但低于++，所以（p+1）需要括号而p++不需要。
指针不能进行乘除运算。
0地址通常表示特殊的地址，它通常是个不能随便碰的地址，你的指针中不应该具有0值。
可以用NULL表示0地址。
指向不同类型的指针不能直接相互赋值。
void*表示不知道指向什么类型的指针。

数组与指针

数组变量其实是特殊的指针，数组变量本身表达地址，如 int a[10];int*p=a;即无需使用
&取地址。但是数组的单元表达的是变量，取地址时仍需要用&。
所以[]既可以对数组使用也可以对指针使用，即p[0]==p,同理，也既可以对指针使用也
可以对数组使用。
实际上数组变量是const的指针，所以数组无法被赋值，就解释了之前学习是所说的不能
将一个数组中每个元素的值赋予另一个数组的各个元素。
因为数组变量已经是const的指针了，所以当出现const int a[]={1，2，3，4，5}；时，
表示的是数组中的每个单元都是const int，所以必须通过初始化进行赋值。因为把数组传
进函数时传递的是地址，所以可以通过将参数设为const来保护数组内的单元的值不会被函
数破环，例 int sum（const int a[],int length);。

关于栈的基本认知

1.栈可以储存具有相同数据类型的多个数据项。
2.栈的操作是受限的，只能往栈中压入数据项（把数据加到栈顶）和从栈中弹出数据项（从同一端移走数据项）。禁止测试或修改不在栈顶的数据项。
即使用栈存储数据元素，对数据元素的“存”和“取”有严格的规定：数据按一定的顺序存储到栈中，当需要调取栈中某数据元素时，需要将在该数
据元素之后进栈的先出栈，该数据元素才能从栈中提取出来。例如栈中存放了 4 个数据元素，进栈的顺序是 A 先进栈，然后 B 进，然后 C 进，最后
D进栈；当需要调取 A 时，首先 D 出栈，然后 C 出栈，然后 B 出栈，最后 A 才能出栈被调用。
3.栈有两种类型顺序栈和链栈。顺序栈通过数组实现，链栈用线性表的链式存储结构实现。
4.栈存储结构调取栈中数据元素时，要避免出现“上溢”和“下溢”的情况：“上溢”：在栈已经存满数据元素的情况下，如果继续向栈内存入数据，
栈存储就会出错。“下溢”：在栈内为空的状态下，如果对栈继续进行取数据的操作，就会出错。但是链栈不会出现”上溢“。

利用递归实现快速排序

近日在课上学习了冒泡排序法，联想到最近学习的函数的递归，
于是利用递归写了一个排序程序，代码如下：
#include<stdio.h>
#define N 10
void sort (int a[], int low, int high);
int split (int a[],int low,int high);

int main()
{
int a[N],i;
for(i=0;i<N;i++)
scanf(“%d”,&a[i]);

sort(a,0,N-1);

for(i=0;i<N;i++)
    printf(" %d",a[i]);
printf("\n");

return 0;

}

void sort (int a[],int low,int high)
{
int middle;

if (low>high)
return;

middle=split(a,low,high);


sort(a,low,middle-1);
sort(a,middle+1,high);

}

int split(int a[],int low,int high)
{
int part_element=a[low];

for(;;){
    
    while(low<high&&part_element<=a[high])
        high--;
    
    if(low>=high)
    break;
    
    a[low]=a[high];
    low++;
    while(low<high&&a[low]<=part_element)
        low++;
        
    if(low>=high)
    break;
    
    a[high]=a[low];
    high--;
}

a[high]=part_element;
return high;

}
代码定义了排序函数sort和分割数列函数split，先通过split得出
一个元素，利用这个元素将函数分割成两段，然后利用递归在两
段进行排序。

一对花括号内设定所包括的元素，如 total = sum ((int []) {5,
3,6,9,11},5);
3.若没有给出表达式return可以在返回类型为void的函数中出
现。
