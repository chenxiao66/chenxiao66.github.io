数组与指针

数组变量其实是特殊的指针，数组变量本身表达地址，如 int a[10];int*p=a;即无需使用
&取地址。但是数组的单元表达的是变量，取地址时仍需要用&。
所以[]既可以对数组使用也可以对指针使用，即p[0]==p,同理，也既可以对指针使用也
可以对数组使用。
实际上数组变量是const的指针，所以数组无法被赋值，就解释了之前学习是所说的不能
将一个数组中每个元素的值赋予另一个数组的各个元素。
因为数组变量已经是const的指针了，所以当出现const int a[]={1，2，3，4，5}；时，
表示的是数组中的每个单元都是const int，所以必须通过初始化进行赋值。因为把数组传
进函数时传递的是地址，所以可以通过将参数设为const来保护数组内的单元的值不会被函
数破环，例 int sum（const int a[],int length);。
