php基础语法学习Ⅳ

1.php中的foreach循环
foreach循环用于遍历数组，其格式为：
foreach（$array as $value)
{
；
}
其中array为数组名称，为每次循环要执行的代码。此时每进行一次
循环，当前元素的值就会被赋值给value，然后下一次循环时数组就会
往后一位。
还有：
foreach（$array as $key =>$value)
{
***;
}
此时每进行一次循环，当前元素的值就会被赋值给value而键会被赋给
key，然后下一次循环时数组就会往后一位。适用于关联数组。
2.php中函数
创建函数的格式如下：
function functionname（）
{
；
}
其中functionname为函数名，为要执行的代码。
注意：函数名称不能以数字开头，应该以字母或下划线开头，而且为了
便于审计代码函数名应提示出它的功能。
若需要函数返回一个值，可以使用return语句。

3.php中的魔术常量
php中有8个魔术常量，他们的值随他们在代码中的位置改变而改变。
这些特殊的常量不区分大小写。
_line_文件中当前的行数。
_file_文件的完整路径和文件名。
dir_文件所在的目录，它等价于dirname（_file)。
_function_函数名称。
_class_类的名称。
_trait_trait的名称。
_method_类的方法名。
_namespace_当前命名空间的名称。
4.php中的命名空间
默认情况下，所有常量、类和函数名都放在全局空间下，我们可以使
用namespace声明命名空间，其语法如下：
namespace MyProject {
const CONNECT_OK = 1;
class Connection { /* … / }
function connect() { / … / }
}
如果一个文件中包含命名空间，它必须在所有其他代码前声明命名空
间。在声明命名空间之前唯一合法的代码是用于定义源文件编码方式
的 declare 语句。所有非 PHP 代码包括空白符都不能出现在命名空
间的声明之前。当然，我们也可以在同一个文件中定义不同的命名空
间代码。将全局的非命名空间中的代码与命名空间中的代码组合在一
起，只能使用大括号形式的语法。全局代码必须用一个不带名称的na
-mespace 语句加上大括号括起来。
命名空间的名字可以使用分层次的方式定义,语法如下：
namespace MyProject\Sub\Level;
const CONNECT_OK = 1;
class Connection { / … / }
function Connect() { / … */ }
PHP中命名空间可以解决以下两类问题：
（1）用户编写的代码与PHP内部的类/函数/常量或第三方类/函数/常
量之间的名字冲突。
（2）为很长的标识符名称(通常是为了缓解第一类问题而定义的)创建
一个别名（或简短）的名称，提高源代码的可读性。
